# Next.js Micro-Frontend Example

This project demonstrates how to use the nxg-sports micro-frontends in a Next.js application.

## Purpose

Integrates the sportsbook micro-frontend using Next.js, showing how to consume shared code from `packages/shared`.

## Setup

1. Install dependencies from the monorepo root:
   ```sh
   pnpm install
   ```
2. Start the development server:
   ```sh
   pnpm dev
   ```
3. Access the app at [http://localhost:3000](http://localhost:3000)
